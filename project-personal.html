Project

AI 이미지 분석 (마스크 & 퍼스널컬러)

프로젝트 기간

개요

마스크 감지·퍼스널 컬러 진단·가상 메이크업을 통합한 AI 뷰티 솔루션입니다.

주요 기능

  1. 마스크 감지

    VGG16 기반 모델로 마스크 착용 여부를 자동 인식

  2. AI 퍼스널 컬러 진단

    이미지 분석으로 봄/여름/가을/겨울 타입을 분류

  3. 가상 메이크업 시뮬레이션

    BeautyGAN 기반으로 다양한 메이크업 스타일을 적용

  4. 사용자 관리 및 이력 저장

    Redis 기반 인증과 마이페이지 이력 관리 기능 제공

아키텍처

역할 및 기여

UI/UX 설계 및 프론트 구현

Vue.js Tailwind CSS

사용자 흐름에 맞춘 화면 설계와 컴포넌트 구조화로 UI 완성도 강화

API 연동 및 백엔드 협업

Spring Boot REST API

프론트-백엔드 간 데이터 계약을 정의하고 API 연동 흐름을 구현

AI 데이터 수집 및 전처리

TensorFlow OpenCV

모델 학습용 이미지 수집과 전처리/증강을 통해 데이터 품질을 확보

퍼스널 컬러 결과 시각화

JavaScript Vue.js

진단 결과와 추천 팔레트를 보기 쉽게 정리한 결과 화면 구성

관리자 기능 개발

Spring Boot Redis

회원 관리 및 인증 흐름을 보조하는 관리자 기능을 추가

프로젝트 후기

향후 개선 방향

  • 실시간 웹캠 기반 분석으로 사용성 확장
  • 가상 메이크업 스타일/브랜드 옵션 확대
  • 모델 경량화와 응답 속도 개선

성장 및 배움

  • AI 모델을 웹 서비스에 연결하는 전체 흐름을 경험
  • 프론트-백엔드 통신 구조와 데이터 처리 과정을 이해
  • 이미지 전처리와 결과 시각화의 중요성을 체감

Project

KBO 리그 통합 데이터 플랫폼

2024.05 ~ 2024.06

개요

흩어진 KBO 리그 정보를 한곳에 모아 경기 데이터, 뉴스, 구단 정보, 커뮤니티를 빠르게 탐색할 수 있는
통합형 야구 정보 플랫폼을 구축했습니다.

주요 기능

  1. 실시간 데이터 통합

    KBO 경기 및 선수 데이터를 통합해 핵심 정보를 한 화면에 제공

  2. 날씨 연동 시각화

    야구장 위치 기반 날씨 정보를 연동해 직관적인 시각화 제공

  3. 뉴스 자동 수집

    최신 KBO 뉴스를 자동 수집해 뉴스 피드로 제공

  4. 구단 인사이트

    구단별 모기업 증권 정보를 차트·지표로 시각화

  5. 커뮤니티

    응원 게시판 커뮤니티 및 댓글/필터링/신고 시스템 제공

  6. 신뢰성 확보

    데이터 출처·오류 모니터링·사용자 피드백 기반으로 신뢰성 강화

아키텍처

주요 역할 및 기여

데이터 수집 자동화 및 DB 연동

BeautifulSoup4 Oracle

BeautifulSoup로 네이버 야구(KBO) 경기 일정 크롤링 로직 구현 및 Oracle DB 자동 적재

이기종 시스템 간 통신 설계

Django Spring MVC

Django(Python)와 Spring(Java) 간 크로스 통신 및 데이터 전송 인터페이스 구현

도메인 맞춤형 스케줄링 최적화

Spring Scheduler Java

Spring Scheduler로 자동 크롤링 구현, 월요일 경기 없음 특성을 반영해 화~일요일만 동작하도록 최적화

UI 기획 및 화면 구성

HTML5 CSS3 JavaScript

야구 일정과 데이터를 직관적으로 확인할 수 있도록 UI 기획 및 레이아웃 구성

주요 화면

주요 화면은 PPT 슬라이드로 확인할 수 있습니다.

프로젝트 후기

향후 개선 방향

  • 경기 흐름 예측 모델 도입으로 인사이트 제공 확대
  • 타자·투수 심화 지표 및 비교 대시보드 고도화
  • 공식 데이터와의 정합성 검증 및 품질 지표 공개
  • 모바일 사용성을 고려한 경량 UI/필터링 개선

성장 및 배움

  • 크롤링 파이프라인 안정화와 구조 변경 대응 경험
  • 이기종 백엔드 통합 및 API 설계의 중요성 체득
  • 데이터 수집 → 정제 → 시각화 전 과정 운영 경험
  • 정보 구조화와 사용자 탐색 흐름 설계 역량 강화
